# AI Study Companion

## Overview

AI Study Companion is an AI-powered learning assistant designed to help students study more effectively. The application allows users to upload study materials, extract content, generate summaries, create quizzes, and improve learning through interactive AI-based tools.

## Problem Statement

Students often spend a significant amount of time reading lengthy notes and textbooks. This project helps learners quickly understand key concepts by automatically summarizing content and generating practice questions from study materials.

## Features

### Current Features

* Upload Study Material (PDF)
* Extract Text from PDF Documents

## Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/AI-Study-Companion.git
```

### Move to Project Directory

```bash
cd AI-Study-Companion
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
